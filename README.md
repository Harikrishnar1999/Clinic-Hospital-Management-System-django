## How to use ?

Step : 1

```bash
  git clone https://github.com/Harikrishnar1999/Clinic-Hospital-Management-System-django.git
```
Step : 2

```bash
  pip install -r requirements.txt 
```
Step : 3

```bash
  python manage.py makemigrations
```
Step : 4

```bash
  python manage.py migrate
```
Step : 5

```bash
  python manage.py runserver
```
